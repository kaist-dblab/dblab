Features and Characteristics

  • A DB-IR tightly-integrated Parallel DBMS: a general purpose database system
    that provides faster search performance than the specialized IR search engines
  • Supporting massive databases (64-bit architecture) (petabyte -> exabyte scale)
  • Providing rich functionalities such as SQL, schemas, and indexes
  • Providing efficient immediate upate
 


Application Areas

  • Large-scale Web search engine like Google, Yahoo!, or Bing
  • Large-scale Web bulletin board like Naver cafe or Daum cafe
  • Large-scale social network service like Twitter
