Research Goal　 
  • Development of a high performance light-weight DBMS for hand-held devices(e.g. mobile-phone, PDA, etc.)

Issues

  • High-performance light-weight DBMS engine
  • Efficient memory management of light-weight DBMS
  • Memory file system management of light-weight DBMS(e.g. flash memory)
  • Crash recovery of light-weight DBMS
  • Communication with servers via computer networks

Application Areas

  • Embedded system
  • Mobile system
  • Ubiquitous system
